# Security Policy

## Important Warning
Do NOT use real passwords, usernames, or credentials that you use for any other service.  
This project is not intended to store or handle sensitive information.
 
## Supported Versions
This project is provided as-is. No security guarantees or ongoing security support are provided.
 
## Reporting a Vulnerability
If you discover a vulnerability, you may report it, but the maintainers make no guarantee of response time or remediation.
 
## User Responsibility
Users are fully responsible for securing their own deployments, servers, databases, and authentication systems.  
The maintainers are not responsible for any data breaches, leaks, or security incidents resulting from the use or misuse of this software.
